断章

发表于 2017-03-05 | 分类于 心情碎片

From：花瓣

2016/07到2016/12的半年，就像人生的断章。
没有写博客。微博删的一干二净。日记断断续续，内容也极为单一。
还好记忆里尚存着关于这段时光的快乐和痛苦，否则我会以为自己不曾活过。

无心学习，看书就困，刚刚睡醒。
索性翻开相册，回顾这悲喜交加的半年。

暑假小忆

超长超待机超开心的暑假。每天在家看看书、做做题，中午帮妈妈准备午饭，下午定时keep锻炼，傍晚散步聊天。在饮食上严格自律，因此体重屡创新低（我至今不知哪来的那么大的毅力）。But暑假快结束时体检被查出轻度贫血，在家人强迫下恢复正常饮食，饭量和体重从此一发不可收拾。

周末一般会选择出行，去定军山天荡山或者新建的诸葛古镇。恰逢夏天瓜果盛，也在晚饭后去了一次葡萄园和蓝莓园。心情是惬意和舒畅的，感慨小县城气候宜人、空气清新、物产丰富以及生活感强，虽然它经济落后。可怎么说呢，经济也是一直发展的，以不断提升的生活水平为感知。

和初中好友匆匆见过一面，约爬云雾山。山间清幽凉爽，有草有花有林有水，有青苔有蘑菇，有陡峭山崖有平行缓坡，行走其间，世间烦恼皆被忘却。我们掐指一算，相识十年，仍有幸相聚回首且真心畅谈，实属不易之缘分。曾经的一行人分分合合聚聚散散，有人走着走着就远了，失了联络，剩下我们这些，且行且珍惜。

闲来无事就想给爸妈露一手，建议妈妈买烤箱，未果。羡慕烘培达人，我以后也要做各种好吃的给我爱的人。

备考时节

不管怎么说已经过去了，而且往往忧患中孕育智慧。可能老天想让我变成一个充满智慧的小仙女吧，嘻嘻(●’◡’●)
总之现在的结果完完全全出乎我意料而且伴随着不可思议的事情发生。我只能感激老天的眷顾，然后珍惜一切，多做好事。

开了个微博小号备考，互粉了不少一起备考的朋友，大家互相鼓励一起进步，总能接收到正能量。特别谢谢他们。

  1. 善良真诚坚毅，永远美好。
  2. 不可有分别心。
  3. 不要在意别人说什么。
  4. 不管结果怎么样，爱你的人始终爱你。
  5. 人生要删繁就简。越简单越幸福。
  6. 不要过分期待某个结果。尽力以后，尽早忘掉。
  7. 得之你幸，失之你命。
